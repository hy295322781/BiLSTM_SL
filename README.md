# BiLSTM_SL
BiLSTM+CRF model for slot filling on ATIS

### Folder Description:
```
BiLSTM_SL
|____ checkpoint	            # path to save checkpoint
|____ data      	            # put ATIS raw data in this folder
|____ output		              # predict results
|____ model.py		            # main code
```

### Usage:
```
python preprocess.py
python main.py
```
